## 函数

```bat
::exit 1 退出并终止执行
::exit /B 1 退出当前bat

call:myfunc
pause

:: 函数
:myfunc
echo this is a bat func
::函数名 %~0
::参数 %~1
echo %~0 %~1c
:: 函数结束标记
goto:eof
```
