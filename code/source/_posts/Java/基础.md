## 基础

### 主文件

```java
// T.java
public class T {
  public static void main(String[] args) {
    System.out.println("Test");
  }
}
// 编译单个文件
javac T.java
// 执行
java T
```

### package

```java
// T.java
package P;
public class T {
  public static void main(String[] args) {
    System.out.println("Test");
  }
}
// 编译包
javac -d . T.java
// 执行
java P.T
```

### import

```java
// S.java
package P;
public class S {
  public void s() {
    System.out.println("S.s");
  }
}
```

```java
// T.java
package P;
import P.S;
public class T {
  public static void main(String[] args) {
    S s = new S();
    s.s();
    System.out.println("Test");
  }
}

// 编译包
javac -d . *.java
// 执行
java P.T
```

### import jar/so

```java
// T.java
import com.sun.jna.Library;
import com.sun.jna.Native;

public class T {
  public interface TestC extends Library {
    // libtestc.so
    TestC INSTANCE = (TestC) Native.loadLibrary("testc", TestC.class);
    void testc();
  }

  public static void main(String[] args) {
    TestC.INSTANCE.testc();
    System.out.println("Test");
  }
}

// linux 使用':'分割, windows使用';'分割
javac WLDecodeJNA.java -cp ".;./jar1;./jar2"
// -cp 指定jar扩展路径
// -Djna.library.path 指定lib扩展路径
java -Djna.library.path="." -cp ".:./jna-3.0.9.jar" WLDecodeJNA
```
