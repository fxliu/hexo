## 数组

```js
function getUrlKey(name) {
    let parames = new RegExp('[?|&]' + name + '=' + '([^&;]+?)(&|#|;|$)').exec(location.href);
    if(parames)
        return decodeURIComponent(parames[1].replace(/\+/g, '%20'));
    return "";
};
```
