## 配置

+ gradle版本号: 工程编译失败时，通常都是gradle配置问题
  + 新建一个空工程
  + 确认build.gradle配置内容
  + 确认gradle\wrapper\gradle-wrapper.properties
+ 备注
  + build.gradle中的版本：gradle插件版本
  + gradle-wrapper.properties中的版本：才是gradle版本
  + [插件版本和版本对应](https://services.gradle.org/distributions/)
