## 系统消息

```bash
# 部分机器不允许发BOOT_COMPLETED消息, root下就好了
adb root
adb shell
# 发送 BOOT_COMPLETED 消息 到指定APP
am broadcast -a android.intent.action.BOOT_COMPLETED -n com.eseid.eid_idcard_svr/com.eseid.eid_idcard_svr.BootBroadcastReceiver
```
